    • K-Modes

    Sumber Data : Disini

    Tipe Data : Categorical

    Source code

    import numpy as np from kmodes.kmodes import KModes

    x = np.genfromtxt('abalone.data.csv', dtype=int, delimiter=',')[:, :-1]

    kmodes_huang = KModes(n_clusters=4, init='Huang', verbose=1) kmodes_huang.fit(x)

    print('abalone.data.csv) centroid:') print(kmodes_huang.cluster_centroids_) print('Final training cost: {}'.format(kmodes_huang.cost_)) print('Training Interations: {}'.format(kmodes_huang.n_iter_))

    • K-Prototype

    Sumber Data : Disini

    Tipe Data : Categorical dan Numerik

    Source code

    ```python import numpy as np from kmodes.kprototypes import KPrototypes

    syms = np.genfromtxt('segmentation.data.csv', dtype=str, delimiter=',')[:, 0] X = np.genfromtxt('segmentation.data.csv', dtype=object, delimiter=',')[:, 1:] X[:, 0] = X[:, 0].astype(int)

    kproto = KPrototypes(n_clusters=2, init='Cao', verbose=2) clusters = kproto.fit_predict(X, categorical=[9, 2])

    print(kproto.cluster_centroids_) print(kproto.cost_) print(kproto.n_iter_)

    for s, c in zip(syms, clusters): print("Symbol: {}, cluster:{}".format(s, c)) ```
